# 更新日志

## v1.0.0 (2024-01-24)

### ✨ 新特性

- 🎉 首次发布 Nova UI 组件库
- ✅ 完成核心组件开发
  - Button 按钮组件
  - Icon 图标组件
  - Input 输入框组件
  - Checkbox 复选框组件
  - Form / FormItem 表单组件
  - Tree 树形控件
  - Upload 上传组件
  - Calendar 日历组件
  - VirtualList 虚拟列表
  - VirtualScrollList 虚拟滚动列表

### 📦 构建系统

- 基于 Vite 构建，支持快速开发和打包
- 使用 pnpm workspace 管理 monorepo
- 完整的 TypeScript 类型定义
- 支持按需加载

### 🧪 测试

- 使用 Vitest 进行单元测试
- 核心组件测试覆盖率达标

### 📝 文档

- 使用 VitePress 构建文档站点
- 提供详细的安装和使用指南
- 每个组件都有完整的 API 文档

### 🔧 工程化

- 集成 ESLint + Prettier 代码规范
- 使用 Husky + lint-staged 进行 Git 提交检查
- 使用 Commitlint 规范提交信息
- 完善的 CI/CD 流程
